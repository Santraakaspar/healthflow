## 🚀 To Run the App:

1. Settings window should be open (if not, run: `start ms-settings:developers`)
2. Toggle "Developer Mode" to ON
3. Click "Yes" if prompted

### Step 2: Run the App

Open PowerShell in the project folder and run:

```powershell

C:\Users\santr\flutter\bin\flutter.bat run -d windows
```

Or if you've opened a new terminal (PATH is updated):


## 🔥 Hot Reload (Amazing Feature!)

While the app is running:
1. Open `lib\main.dart` or any screen file
2. Make changes to the code
3. Press `r` in the terminal (or save the file)
4. See changes **instantly** without restarting!

## 🐛 Troubleshooting:

### "Developer Mode required"
- Make sure you enabled Developer Mode in Settings
- Restart your computer if needed
- Try running the command again

### "No devices found"
Run: `flutter devices`
You should see "Windows (desktop)"

### Build errors
Run: `flutter clean` then `flutter pub get` then try again
